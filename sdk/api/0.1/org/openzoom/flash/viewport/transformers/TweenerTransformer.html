Packageorg.openzoom.flash.viewport.transformers
Classpublic class TweenerTransformer
ImplementsIViewportTransformer

TweenerTransformer is an implementation of IViewportTransformer based on the fantastic animation library Tweener. It let's you specify the duration and easing of the animation.
